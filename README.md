众所周知，ThinkPad一直以优秀的工业设计、优异的可靠性和丰富的可拓展性被广大电子爱好者们所喜爱，其中继承自蓝色巨人IBM的ThinkPad经典7行键盘尤为受到追捧。

新款孤岛键盘更换经典的7行键盘之后存在一些例如键盘大写灯不亮等兼容性问题，并且部分键位会发生改变，因此刷写客制化EC重新映射键位成为必须要进行的步骤。

得益于hamishcoleman的thinkpad-ec项目（https://github.com/hamishcoleman/thinkpad-ec）的贡献，更改ThinkPad EC变得简单起来。但是由于这个项目主要是在Linux环境下构建的，因此对一部分不熟悉Linux操作或者缺少Linux运行环境的爱好者来说变得不便。

此项目基于全新安装的Debian11 Linux操作系统环境将所有支持的ThinkPad型号进行了构建，使得所有人都可以方便地获取ISO镜像成品。

现在，只需要下载你的机器所对应的镜像，使用Rufus或者Etcher等磁盘烧录工具将其烧录到U盘之中就可以优雅地进行ThinkPad的EC刷写！
